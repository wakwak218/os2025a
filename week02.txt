cd 폴더명 (change directary) 현재 디렉터리 바꾸기
cd .. 상위폴더
ls (list) 디렉터리 내용 출력
touch 파일 생성
cat 파일명 텍스트 파일 내용 띄우기

git init 폴더 깃 연결
git status 작업 현 상태
git log 깃 기록 보기
git config --global user.email "개인이메일주소" 
git config --global user.name "id" 
git config --list 구성목록 띄우기
config는 환경설정(딱 한 번만 하면 됨)

git add 파일명
git add .
git commit -m "커밋에 남길 메세지" 커밋
git checkout 이전에 커밋한 작업으로 돌아가기 가능
